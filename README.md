# SEED Labs – Secret-Key Encryption Lab Guide

A complete beginner-friendly study guide for the SEED Secret-Key Encryption Lab, covering all tasks with deep explanations and visual diagrams.

## What's inside

- **Task 1** — Caesar cipher: encryption, brute force attack, frequency analysis
- **Task 2** — Modern symmetric encryption: ciphers, password-based encryption, padding, ECB/CBC modes, weak DES keys, the ECB Penguin
- **Task 3** — Error propagation across ECB, CBC, CFB, and OFB modes
- **Task 4** — IV uniqueness and the OFB known-plaintext attack

## Built with

Pure HTML + CSS + SVG — no frameworks, no dependencies, works fully offline.
